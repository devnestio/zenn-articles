## MIME Type Lookup — 無料オンラインツール

[MIME Type Lookup](https://mime-type-lookup-abi.pages.dev) は、ファイル拡張子からMIMEタイプを検索、またはMIMEタイプから拡張子を逆引きできる無料ツールです。400種以上対応。

## 主な機能

- **拡張子 → MIME** — `.html` と入力すれば即座に `text/html` を表示
- **MIME → 拡張子** — `image/jpeg` を入力すると `jpg`、`jpeg` を表示
- **400種以上** — text、image、audio、video、application、font、model、message、multipartに対応
- **カテゴリフィルター** — タイプカテゴリ別に絞り込み
- **全文検索** — 拡張子やMIME文字列の一部で検索
- **クイックチップ** — よく使うタイプをワンクリックで検索
- **コピーボタン** — 各結果に即コピーボタン

## 対応カテゴリ

| カテゴリ | 例 |
|---------|---|
| text | html, css, js, json, csv, md, yaml |
| image | jpg, png, webp, avif, svg, gif |
| audio | mp3, wav, flac, ogg, aac, opus |
| video | mp4, webm, mkv, mov, avi |
| application | pdf, zip, wasm, docx, epub |
| font | woff2, ttf, otf |
| model | stl, obj, gltf, glb |
