## Character Counter Proとは

[Character Counter Pro](https://character-counter-acj.pages.dev) は、テキストを入力するだけでリアルタイムに文字数・単語数・文章数などを分析できる無料ツールです。

## 主な機能

- **8種のコア統計**: 文字数・スペースなし文字数・単語数・文章数・段落数・行数・平均単語長・ユニーク単語数
- **SNSリミットチェック**: Twitter/X (280)・LinkedIn投稿 (3,000)・Instagram Bio (150)・Instagram Caption (2,200)・YouTube Title (100)・Meta Description (160)・SMS (160) など10種のプログレスバー表示
- **読書時間推定**: 遅い (150wpm)・平均 (238wpm)・速い (400wpm) の3段階
- **頻出単語マップ**: ストップワードを除いた上位10単語とバー表示

## 使い方

テキストエリアに文章を貼り付けるだけでリアルタイムに全統計が更新されます。
