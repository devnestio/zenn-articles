## Makefile Generatorとは

[Makefile Generator](https://makefile-generator-ach.pages.dev) は、Go・Python・Node.js・Rust・C/C++向けのMakefileをブラウザ上で即座に生成できる無料ツールです。

## 主な機能

- **6言語テンプレート**: Generic / Go / Python / Node.js / Rust / C/C++
- **13種のターゲット**: build / test / clean / lint / fmt / run / install / docker / docker-push / release / ci / help
- **クロスコンパイル対応**: Go向けにlinux/darwin/windowsの一括ビルドコマンドを生成
- **Docker統合**: IMAGE/TAGの変数管理とdocker-pushターゲット
- **コピー＆ダウンロード**: 生成したMakefileをワンクリックで取得

## 使い方

1. 言語を選択（Go / Python / Node.js / Rust / C/C++）
2. プロジェクト名を入力
3. 含めたいターゲットをチェック
4. 「Generate Makefile」をクリック
5. コピーまたはダウンロード
