## Number Formatterとは

[Number Formatter](https://number-formatter-ack.pages.dev) は、1つの数値を20種類以上のフォーマットに即座に変換できる無料ツールです。

## 主な機能

- **ロケール対応**: en-US / de-DE / fr-FR / ja-JP / zh-CN / ko-KR / ar-SA など15ロケール
- **通貨表示**: USD・EUR・GBP・JPY・CNY・KRW・INR など15通貨
- **科学的記数法**: JS形式 (1.23e+6) と Intl形式
- **コンパクト表示**: 1M / 1 million 形式
- **バイト単位**: SI (KB/MB/GB/TB) と IEC (KiB/MiB/GiB/TiB)
- **進数変換**: 16進・2進・8進
- **その他**: 序数 (1st/2nd)・ローマ数字・固定小数点・有効数字

## クイックプリセット

0 / 1K / 1M / 1 GiB / π / MAX_SAFE_INT など8種のプリセットボタンで素早くテストできます。
