## 一言で言うと

アップロード不要・完全ブラウザ完結で、画像のリサイズ・クロップ・回転・フリップができるツールです。PNG / JPEG / WebP 出力と画質コントロールにも対応しています。

👉 https://image-resizer-abs.pages.dev

devnestio（ https://devnestio.pages.dev ）は、英語圏の開発者向けに無料ツールを少しずつ公開しているサイトです。今回はその中のひとつ、Image Resizerについてbuilding-in-publicで書いてみます。

## なぜ作ったのか

「ちょっと画像を小さくしたい」だけなのに、Photoshop を起動したり、オンラインサービスにアップロードしたりするのが面倒でした。Canvas API を使えばブラウザ内で完結できるので、サーバーに画像を送らずに済むツールを作りました。

## 主な機能

- **リサイズ**: 幅・高さをピクセル指定、アスペクト比ロック対応
- **スケールスライダー**: 元サイズの 10%〜200% でざっくり変更
- **クロップ**: 任意の矩形領域を切り取り
- **回転・フリップ**: 任意の角度回転、水平・垂直フリップ
- **出力フォーマット**: PNG / JPEG / WebP
- **品質コントロール**: JPEG / WebP は 1〜100% で指定

## 技術的なポイント

Canvas の `drawImage` と `toBlob` を組み合わせることで、変換処理をすべてクライアントサイドで実行しています。画像データがサーバーに送信されることは一切なく、プライバシーフレンドリーな設計になっています。

## まとめ

シンプルだからこそ、起動の速さと操作の軽快さにこだわりました。「すぐ小さくしたい」という需要に応えるツールです。インストール不要で今すぐ使えます。
