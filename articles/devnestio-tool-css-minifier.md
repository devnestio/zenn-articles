## 主な機能

- **圧縮（Minify）** — コメント削除、不要な空白除去、`}`直前の`;`削除
- **整形（Beautify）** — 圧縮済みCSSを読みやすい形式に再整形
- **圧縮統計** — 圧縮前後のバイト数、削減バイト数、圧縮率
- **ルール数カウント** — CSSルールの総数を表示
- **オプション** — コメント保持、最後のセミコロン保持
- **入出力スワップ** — ワンクリックで入出力を入れ替え
- **サンプル** — シンプルスタイル, メディアクエリ, CSS変数, キーフレーム, 圧縮済みCSS

## 圧縮の仕組み

```css
/* 圧縮前（287バイト） */
body {
  margin: 0;
  padding: 0;
  background-color: #ffffff;
  font-family: Arial, sans-serif;
}

/* 圧縮後（64バイト） — 77.7%削減 */
body{margin:0;padding:0;background-color:#fff;font-family:Arial,sans-serif}
```

圧縮ステップ：
1. `/* ... */` ブロックコメントを削除
2. `{`, `}`, `:`, `;`, `,` 周辺の空白を削除
3. 連続した空白を1つにまとめる
4. `}` 直前の `;` を削除（各ルールで1バイト節約）

---

## 整形の仕組み

フルCSSパーサーは使わず、文字単位のトークナイザーを実装：

- `{` → 改行 + インデント増加
- `}` → インデント減少 + 改行
- `;` → 改行 + 現在のインデントを保持

ベンダープレフィックスや `calc()` 式でも正しく動作します。
